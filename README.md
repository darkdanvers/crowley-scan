## Install requirements

```
pip install -r requirements.txt
```

## Usage

```
Usage: crowley.py [OPTIONS]

Options:
  --dork TEXT
  --help       Show this message and exit.
```

## Examples

```
python crowley.py --dork "news.php?id="
```
